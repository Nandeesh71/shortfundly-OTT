# Shortfundly OTT — Product Overview

Shortfundly is an Indian OTT (Over-The-Top) streaming platform focused on short films, movies, and web series. It operates at `web.shortfundly.com`.

## Core Purpose
- Discover, watch, and submit short films and movies online
- Supports free and premium content tiers
- Targets Indian audiences with geo-aware routing and regional language recommendations

## Key Features
- Content browsing: Films, Movies, Web Series, Shorts, Upcoming, Award-winning, Top Rated
- User accounts: Email/Password, Phone+OTP, Google, Facebook, Apple OAuth
- Watchlist, Favorites, Continue Watching, Like/Unlike
- Push notifications via OneSignal and iZooto
- Video playback via JWPlayer + Upremium CDN
- Payment integrations: Razorpay, PhonePe, Stripe (dns-prefetched)
- PWA with standalone display mode

## App Distribution
Available on: Web, Google Play, Apple App Store, Amazon, Jio, Android TV, Cloud TV, Indus
